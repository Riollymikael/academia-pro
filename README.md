## 📌 Sobre o Projeto

Uma landing page moderna, dinâmica e 100% responsiva para a **Academia Pro**, criada para apresentar planos de treino, estrutura da academia, modalidades oferecidas e captação de novos alunos através de chamadas estratégicas para ação (CTA).

### ✨ Diferenciais do Layout:
* ⚡ **Hero Section:** Banner de alto impacto visual com foco em conversão.
* 🥊 **Modalidades & Treinos:** Apresentação clara das aulas e equipamentos.
* 💳 **Tabela de Planos:** Exibição de preços e benefícios de forma transparente.
* 💬 **Depoimentos & Transformações:** Prova social de alunos para aumentar a credibilidade.
* 📱 **Layout Responsivo:** Adaptado perfeitamente para celulares, tablets e computadores.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização moderna com Flexbox, CSS Grid e efeitos visuais.
* **JavaScript:** Interatividade e dinamismo nas seções.
